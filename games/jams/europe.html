This game is not a game about Europe. This game is a game about games about Europe. Embody a board game creator specialized on Europe and fulfill orders from demanding (and biased) clients. Investigate their opinions on geopolitics in Europe, and deliver them the perfect game.


L'Europe sur un Plateau was created in 48 hours as a submission to the Scientific Game Jam - Lyon #3, which had 12 teams work on games based on some of the contestants' theses. This game was created from a thesis dealing with the depiction of Europe and its geopolitical issues in board games. It was awarded the Grand Prix and the Public's Favorite prize of the jam. I was part of a 8-people team, as one of the two developers. I also took part in the game design discussions, and helped design the computer UI of the game.
